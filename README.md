# 📊 Internship Completion Analysis 

## 🔷 Project Overview
This project focuses on analyzing internship data to identify key factors that influence intern completion and dropout rates. The goal is to uncover meaningful insights and visualize patterns that help improve intern success.

---

## 🎯 Objective
To analyze and visualize internship completion rates and identify the major factors affecting intern performance, such as department, internship duration, and mentor interaction.

---

## 📁 Dataset
The dataset includes the following features:

- **Intern_ID** – Unique identifier for each intern  
- **Department** – Department assigned (Marketing, IT, HR, Finance)  
- **Duration_Weeks** – Internship duration in weeks  
- **Mentor_Meetings** – Number of mentor interactions  
- **Enrolled** – Indicates if the intern enrolled (1 = Yes)  
- **Completed** – Indicates successful completion (1 = Yes)  
- **Dropped** – Indicates dropout (1 = Yes)  

---

##  Dashboard Features

### 📌 KPI Cards
- Total Interns  
- Completion Rate  
- Dropout Rate  

### 📌 Department Analysis
- Completion trends across departments  

### 📌 Duration Analysis
- Impact of internship length on completion  

### 📌 Mentor Interaction Analysis
- Relationship between mentor meetings and success  

### 📌 Completion vs Dropout Distribution
- Overall performance overview  

---

## 🔍 Key Insights

- Interns with higher mentor interaction show significantly better completion rates.  
- Shorter internship durations (6–8 weeks) tend to have higher success rates.  
- Some departments exhibit higher dropout rates, indicating a need for improved support.  
- Mentor engagement plays a critical role in intern retention and performance.
